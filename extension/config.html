How The Final Bet Works

  1. Start Betting Window - Viewers have ~60 seconds to place bets
  2. Close Betting - When ready to start the game/challenge
  3. Play Game - Game happens (variable length), extension stays open
  4. Resolve Manually - Mark as SUCCESS or FAILED when game ends
  5. Payouts Distributed - Winners automatically receive their Bits
